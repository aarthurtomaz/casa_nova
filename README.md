# Site - Chá de Casa Nova

Este projeto foi desenvolvido como **freelance** com o objetivo de divulgar um **chá de casa nova**, permitindo que convidados tenham acesso às informações do evento de forma simples e organizada.

🔗 Acesse o site:  
https://aarthurtomaz.github.io/casa_nova/

---

# Objetivo do projeto

O objetivo deste site foi criar uma página web simples e direta para compartilhar informações importantes sobre o evento, como:

- Informações do chá de casa nova
- Detalhes do evento
- Local e data
- Forma de participação dos convidados

A proposta foi desenvolver uma página **leve, acessível e fácil de compartilhar**, permitindo que o anfitrião envie o link diretamente para os convidados.

---

# 💼 Contexto do projeto

Este foi um **trabalho freelance**, desenvolvido para atender a necessidade de divulgação de um evento familiar.

O foco do projeto foi:

- Criar um site simples e visualmente agradável
- Facilitar o acesso às informações do evento
- Publicar rapidamente a página online
- Utilizar tecnologias web básicas para uma solução eficiente
